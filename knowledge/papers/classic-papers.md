# Reading List & Dependencies

This list includes the core neurotechnology frameworks, software dependencies, and validation foundations that influence VIREON's design:

- **BrainFlow**: A robust library to obtain, parse, and analyze EEG, EMG, ECG, and other kinds of physiological data across diverse boards.
- **OpenBCI**: Open-source brain-computer interfaces. VIREON often targets OpenBCI-compatible telemetry and hardware limits for hardware-in-the-loop (HIL) simulations.
- **MNE-Python**: The premier open-source Python package for exploring, visualizing, and analyzing human neurophysiological data.
- **PhysioNet**: The Research Resource for Complex Physiologic Signals (primary source of VIREON validation datasets like CHB-MIT and Sleep-EDF).
